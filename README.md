### 🌟 What is BanglaHost?

**BanglaHost** is a modern, high-performance local development environment built specifically for Windows developers. It eliminates the hassle of configuring complex runtimes and databases, providing a **one-click experience** for PHP, Python, Node.js, and more.

Built with the cutting-edge **.NET 8** and **WinUI 3 (Windows App SDK)**, it offers a native, fluid interface that feels right at home on Windows 10 and 11.

---

## ⚡ Key Features at a Glance

### 🌍 Multi-Runtime Support
Switch between language versions instantly without ever touching your system Environment Variables.
- **PHP:** 7.4, 8.0, 8.1, 8.2, 8.3+
- **Python:** Automated venv management for Flask, Django, FastAPI.
- **Node.js:** Integrated version manager for Express, Next.js, and React.
- **Go & Rust:** Native support with automated reverse proxying via Nginx.

### 🗄️ Comprehensive Database Suite
Isolated, pre-configured database instances that run only when you need them.
- **Relational:** MySQL, MariaDB, PostgreSQL.
- **NoSQL:** MongoDB, Redis.
- **Embedded:** SQLite (Pre-optimized).

### 🛠️ Developer Power Tools
- **Mailpit:** Catch and debug outgoing emails locally.
- **AI Assistant:** Integrated **Ollama** support to run local LLMs (Llama 3, Mistral) for coding help.
- **Networking:** Automated **SSL/HTTPS** and **Cloudflare Tunnel** for instant secure public sharing.
- **Task Automation:** Built-in **Cron Scheduler**, WP-CLI, Composer, and Docker controls.

## 🛠️ Tech Stack & Architecture

- **Core Logic:** C# / .NET 8 (Modular Service-Oriented Architecture)
- **Frontend:** WinUI 3 / XAML (Fluent Design System)
- **Pattern:** MVVM (CommunityToolkit.Mvvm)
- **Web Servers:** Custom-tuned Nginx & Apache binaries for Windows.
- **Process Management:** High-performance async service controllers.

## 🚀 Getting Started

### 1. Installation
The easiest way to install is via the **Microsoft Store**:
- [Get BanglaHost on Microsoft Store](https://apps.microsoft.com/store/detail/9MWFKR8D8318)

Alternatively, download the latest `.msix` or `.exe` from our [Releases](https://github.com/mohammad-sheikh-shahinur-rahman/BanglaHost/releases) page.

### 2. Quick Setup
1. Launch BanglaHost.
2. Go to **Sites** -> **Add New Site**.
3. Choose your folder and runtime.
4. Hit **Start** and visit your `.test` domain!
