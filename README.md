# 🚀 BanglaHost

**BanglaHost** is the ultimate local web development environment for Windows, designed to be modern, fast, and feature-rich. Built with **.NET 8** and **WinUI 3**, it provides a seamless experience for managing local development sites with support for multiple languages and databases.

## 🌟 Key Features

-   **Multi-Language Support:** PHP, Python, Node.js, Go, and Rust.
-   **Comprehensive Databases:** MySQL, MariaDB, PostgreSQL, MongoDB, SQLite, and Redis.
-   **Web Servers:** Includes both Nginx and Apache out of the box.
-   **Elegant Dashboard:** Manage all your sites from a modern WinUI 3 interface.
-   **Runtime Manager:** Switch between different language versions instantly without polluting your system PATH.
-   **Networking & Security:** Automated SSL certificates and Cloudflare Tunnel integration for secure sharing.
-   **Developer Tools:** Built-in Terminal, Logs viewer, Git integration, Composer, WP-CLI, and Docker support.
-   **Advanced Features:** Mailpit for local email testing, Cron Scheduler, API Client, and Site Cloning.
-   **AI Assistant:** Integrated Ollama support for running local AI models.

---

## 🛠️ Built With

-   **Frontend:** WinUI 3 (Windows App SDK)
-   **Backend Logic:** .NET 8
-   **Pattern:** MVVM (Model-View-ViewModel)
-   **Architecture:** Modular C# services for engine management and UAC elevation.

## 🚀 Getting Started

### Installation

1.  **Microsoft Store (Recommended):** Get it from the [Microsoft Store](https://apps.microsoft.com/store/detail/9MWFKR8D8318).
2.  **Manual:** Download the `.msix` package from the [Releases](https://github.com/mohammad-sheikh-shahinur-rahman/banglahost-dev/releases) page and install it.

### System Requirements

-   **OS:** Windows 10 (1809+) or Windows 11.
-   **Architecture:** x64 or ARM64.
-   **RAM:** 4GB minimum (8GB recommended).
