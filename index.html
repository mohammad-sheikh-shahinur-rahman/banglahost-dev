BanglaHost Documentation

The ultimate local web development environment for Windows. Built with .NET 8 and WinUI 3.

Get from Microsoft Store Visit Website Quick Start
Version 1.0.0 Platform: Windows

Project Dashboard

Manage all your local sites in one elegant WinUI 3 dashboard.

Runtime Manager

Instantly install and switch between PHP, Python, Node, Go, and Rust.

Databases

Built-in MySQL, PostgreSQL, MongoDB, SQLite, and Redis.

Features at a Glance

  • Multi-Language: PHP, Python, Node.js, Go, Rust
  • Databases: MySQL, MariaDB, PostgreSQL, MongoDB, Redis, SQLite
  • Web Servers: Nginx & Apache included out of the box
  • Tools: Git, Composer, WP-CLI, Docker support
  • Advanced: Mailpit, Cron Scheduler, API Client
  • AI Assistant: Built-in Ollama integration for local AI
  • Networking: Auto-SSL & Cloudflare Tunnels
  • Management: Backup, Restore, and 1-Click Site Clone
  • Monitoring: Real-time System Metrics & Performance Monitor
  • Architecture: Modern .NET 8 + WinUI 3 with MVVM pattern
